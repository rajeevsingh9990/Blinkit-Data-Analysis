# 🛒 Blinkit Sales Analysis using Python

## 📌 Project Overview

This project analyzes Blinkit's sales data to uncover business insights related to product performance, outlet characteristics, customer preferences, and overall sales trends. Using Python and data visualization techniques, the project transforms raw sales data into meaningful insights that support business decision-making.

## 🎯 Objective

The objective of this project is to:

* Analyze overall sales performance.
* Identify top-performing product categories.
* Compare sales across outlet types and locations.
* Understand the impact of fat content, outlet size, and establishment year on sales.
* Generate actionable business insights through visualizations.

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Dataset Information

The dataset contains **8,523 records** and **12 attributes**, including:

* Item Type
* Item Fat Content
* Sales
* Rating
* Outlet Type
* Outlet Size
* Outlet Location Type
* Outlet Establishment Year
* Item Visibility
* Item Weight

## 🧹 Data Cleaning

Data preprocessing steps performed:

* Checked dataset structure and data types.
* Standardized inconsistent values in the **Item Fat Content** column.
* Replaced:

  * LF → Low Fat
  * low fat → Low Fat
  * reg → Regular
* Verified data quality before analysis.

## 📊 Key Performance Indicators (KPIs)

The following KPIs were calculated:

* Total Sales
* Average Sales
* Number of Items Sold
* Average Product Rating

## 📈 Visualizations Created

### 1. Total Sales by Fat Content

* Compared sales contribution from Low Fat and Regular products.
* Visualized using a Pie Chart.

### 2. Total Sales by Item Type

* Identified top-performing product categories.
* Visualized using a Bar Chart.

### 3. Fat Content by Outlet for Total Sales

* Compared product sales across outlet tiers.
* Visualized using a Grouped Bar Chart.

### 4. Total Sales by Outlet Establishment Year

* Analyzed sales performance based on outlet age.
* Visualized using a Line Chart.

### 5. Sales by Outlet Size

* Compared sales distribution across outlet sizes.
* Visualized using a Pie Chart.

### 6. Sales by Outlet Location Type

* Evaluated sales performance across different outlet locations.
* Visualized using a Horizontal Bar Chart.

## 🔍 Business Insights

* Certain product categories contribute significantly more revenue than others.
* Outlet location and size influence overall sales performance.
* Product fat content impacts customer purchasing patterns.
* Established outlets tend to generate stronger sales performance.
* Sales distribution varies across outlet tiers and locations.
